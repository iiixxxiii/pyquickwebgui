Welcome to pyquickwebgui's documentation!
=========================================

`pyquickwebgui` 是一个轻量级 Python 库，用于快速构建基于 Web 的 GUI 应用程序。它集成了 Web 框架、GUI 控件和插件系统，适用于桌面工具开发、快速原型设计等场景。
